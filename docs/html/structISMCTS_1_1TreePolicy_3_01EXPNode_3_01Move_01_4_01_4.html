Detailed Description

template<class Move>
struct ISMCTS::TreePolicy< EXPNode< Move > >

EXP3: Exponential weight algorithm for Exploitation and Exploration

The EXP3 method uses and updates a non-uniform distribution of probabilities over the nodes to select one, skewing this distribution over time in favour of nodes with better expected rewards.
